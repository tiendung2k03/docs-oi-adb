API Reference

MyADB

Lớp chính của OIADB, cung cấp giao diện để tương tác với ADB.

__init__

Constructor 
def __init__(self, device_id=None, auto_install_adb=True, adb_path=None)

Khởi tạo đối tượng MyADB.

Tham số

Tên Kiểu Mặc định Mô tả
device_id str None ID của thiết bị để kết nối. Nếu None, sẽ sử dụng thiết bị đầu tiên được tìm thấy.
auto_install_adb bool True Tự động tải xuống và cài đặt ADB nếu cần.
adb_path str None Đường dẫn đến ADB đã cài đặt. Nếu None, sẽ tìm kiếm trong PATH hoặc tự động cài đặt.

Ví dụ

from oiadb import MyADB

# Khởi tạo với thiết bị mặc định
adb = MyADB()

# Khởi tạo với thiết bị cụ thể
adb = MyADB(device_id="emulator-5554")

# Tắt tự động cài đặt ADB
adb = MyADB(auto_install_adb=False)

# Chỉ định đường dẫn ADB
adb = MyADB(adb_path="/path/to/adb")

get_devices

Method 
def get_devices(self)

Lấy danh sách các thiết bị đã kết nối.

Trả về

list: Danh sách các ID thiết bị đã kết nối.

Ví dụ

devices = adb.get_devices()
print(devices)  # ['emulator-5554', 'device_id_2', ...]

get_device_info

Method 
def get_device_info(self)

Lấy thông tin chi tiết về thiết bị hiện tại.

Trả về

dict: Từ điển chứa thông tin thiết bị (model, manufacturer, android_version, etc.).

Ví dụ

device_info = adb.get_device_info()
print(f"Model: {device_info['model']}")
print(f"Android version: {device_info['android_version']}")

reboot_device

Method 
def reboot_device(self, mode=None)

Khởi động lại thiết bị.

Tham số

Tên Kiểu Mặc định Mô tả
mode str None Chế độ khởi động lại. Có thể là None (khởi động bình thường), 'bootloader', hoặc 'recovery'.

Trả về

bool: True nếu thành công, False nếu thất bại.

Ví dụ

# Khởi động lại bình thường
adb.reboot_device()

# Khởi động vào bootloader
adb.reboot_device(mode='bootloader')

# Khởi động vào recovery
adb.reboot_device(mode='recovery')

execute_command

Method 
def execute_command(self, command, shell=False, timeout=None)

Thực thi lệnh ADB.

Tham số

Tên Kiểu Mặc định Mô tả
command str or list Required Lệnh ADB để thực thi. Có thể là chuỗi hoặc danh sách các đối số.
shell bool False Nếu True, thực thi lệnh trong shell của thiết bị.
timeout float None Thời gian chờ tối đa (giây) cho lệnh. None có nghĩa là không có thời gian chờ.

Trả về

str: Kết quả đầu ra của lệnh.

Ví dụ

# Thực thi lệnh ADB cơ bản
result = adb.execute_command("devices")
print(result)

# Thực thi lệnh shell
result = adb.execute_command("ls /sdcard", shell=True)
print(result)

# Thực thi với thời gian chờ
result = adb.execute_command("pull /sdcard/large_file.zip", timeout=60)
print(result)

Interaction

Module này cung cấp các phương thức để tương tác cơ bản với thiết bị Android.

tap

Function 
def tap(x, y)

Nhấn vào tọa độ màn hình.

Tham số

Tên Kiểu Mô tả
x int Tọa độ X trên màn hình.
y int Tọa độ Y trên màn hình.

Trả về

bool: True nếu thành công, False nếu thất bại.

Ví dụ

from oiadb.commands import interaction

# Nhấn vào tọa độ (500, 800) trên màn hình
interaction.tap(500, 800)

swipe

Function 
def swipe(start_x, start_y, end_x, end_y, duration=300)

Vuốt từ điểm bắt đầu đến điểm kết thúc.

Tham số

Tên Kiểu Mặc định Mô tả
start_x int Required Tọa độ X bắt đầu.
start_y int Required Tọa độ Y bắt đầu.
end_x int Required Tọa độ X kết thúc.
end_y int Required Tọa độ Y kết thúc.
duration int 300 Thời gian vuốt (ms).

Trả về

bool: True nếu thành công, False nếu thất bại.

Ví dụ

# Vuốt từ trái sang phải
interaction.swipe(100, 500, 900, 500)

# Vuốt từ trên xuống dưới (cuộn xuống)
interaction.swipe(500, 300, 500, 1500)

# Vuốt chậm (1 giây)
interaction.swipe(100, 500, 900, 500, duration=1000)

Image Interaction

Module này cung cấp các phương thức để tương tác với thiết bị dựa trên nhận diện hình ảnh.

find_image

Function 
def find_image(template_path, threshold=0.8, region=None, scale_range=(0.8, 1.2), scale_steps=5, rotation_range=(0, 0), rotation_steps=1, use_gray=True, use_canny=False)

Tìm hình ảnh mẫu trên màn hình thiết bị.

Tham số

Tên Kiểu Mặc định Mô tả
template_path str Required Đường dẫn đến hình ảnh mẫu.
threshold float 0.8 Ngưỡng tương đồng (0.0 - 1.0).
region tuple None Vùng tìm kiếm (x, y, width, height). None = toàn màn hình.
scale_range tuple (0.8, 1.2) Phạm vi tỷ lệ để tìm kiếm.
scale_steps int 5 Số bước tỷ lệ.
rotation_range tuple (0, 0) Phạm vi góc xoay (độ).
rotation_steps int 1 Số bước góc xoay.
use_gray bool True Sử dụng ảnh xám để tăng tốc độ.
use_canny bool False Sử dụng phát hiện cạnh Canny cho hình ảnh mờ.

Trả về

tuple or None: (x, y, confidence) nếu tìm thấy, None nếu không tìm thấy.

Ví dụ

from oiadb.commands import image_interaction

# Tìm hình ảnh cơ bản
result = image_interaction.find_image("/path/to/button.png")
if result:
    x, y, confidence = result
    print(f"Đã tìm thấy hình ảnh tại ({x}, {y}) với độ tương đồng {confidence}")

# Tìm với ngưỡng tương đồng cao hơn
result = image_interaction.find_image("/path/to/icon.png", threshold=0.9)

# Tìm trong vùng cụ thể
result = image_interaction.find_image("/path/to/element.png", region=(100, 200, 400, 300))

# Tìm với nhiều tỷ lệ và góc xoay
result = image_interaction.find_image(
    "/path/to/template.png",
    scale_range=(0.5, 1.5),
    scale_steps=10,
    rotation_range=(-30, 30),
    rotation_steps=6
)
