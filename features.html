Tính năng OIADB

Tính năng cơ bản

OIADB là một thư viện Python wrapper cho Android Debug Bridge (ADB) với nhiều tính năng mạnh mẽ, giúp đơn giản hóa việc tương tác với thiết bị Android thông qua ADB.

Tự động cài đặt ADB

OIADB tự động phát hiện và cài đặt ADB nếu cần, không yêu cầu cài đặt thủ công. Thư viện cũng tự động cài đặt và quản lý oiadb-server.apk trên thiết bị Android của bạn.

from oiadb import MyADB

# ADB sẽ được tự động tải xuống và cài đặt nếu cần
adb = MyADB()

# Hoặc tắt tự động cài đặt ADB nếu bạn đã cài đặt
adb = MyADB(auto_install_adb=False)

Quản lý thiết bị

Dễ dàng liệt kê, kết nối và quản lý nhiều thiết bị Android, bao gồm cả kết nối không dây.

# Liệt kê các thiết bị đã kết nối
devices = adb.get_devices()
print(devices)

# Lấy thông tin thiết bị
device_info = adb.get_device_info()
print(device_info)

# Kết nối qua mạng không dây
from oiadb.commands import connect
connect.connect_over_network("192.168.1.100", 5555)

# Khởi động lại thiết bị
adb.reboot_device()

Quản lý ứng dụng

Cài đặt, gỡ cài đặt, khởi động và quản lý các ứng dụng trên thiết bị Android.

from oiadb.commands import apps

# Cài đặt ứng dụng
apps.install_app("/path/to/app.apk")

# Gỡ cài đặt ứng dụng
apps.uninstall_app("com.example.app")

# Khởi động ứng dụng
apps.start_app("com.example.app")

# Dừng ứng dụng
apps.stop_app("com.example.app")

# Xóa dữ liệu ứng dụng
apps.clear_app_data("com.example.app")

# Lấy danh sách ứng dụng đang chạy
running_apps = apps.get_running_apps()

Tương tác cơ bản

Thực hiện các thao tác cơ bản như nhấn, vuốt, nhập văn bản và các thao tác phím.

from oiadb.commands import interaction

# Nhấn vào tọa độ màn hình
interaction.tap(500, 500)

# Vuốt màn hình
interaction.swipe(100, 500, 600, 500)

# Nhập văn bản
interaction.text_input("Hello World")

# Nhấn phím Home
interaction.home()

# Nhấn phím Back
interaction.back()

# Nhấn phím Recent Apps
interaction.recent_apps()

Nhận diện hình ảnh

Tính năng nhận diện hình ảnh của OIADB cho phép bạn tìm kiếm và tương tác với các phần tử trên màn hình dựa trên hình ảnh mẫu, mà không cần biết tọa độ cố định hoặc cấu trúc UI.

Tìm hình ảnh

Tìm kiếm hình ảnh mẫu trên màn hình thiết bị và trả về tọa độ nếu tìm thấy.

from oiadb.commands import image_interaction

# Tìm hình ảnh đơn
result = image_interaction.find_image(
    template_path="/path/to/template.png",
    threshold=0.8,  # Ngưỡng tương đồng (0.0 - 1.0)
    region=None,    # Vùng tìm kiếm (x, y, width, height), None = toàn màn hình
    scale_range=(0.8, 1.2),  # Phạm vi tỷ lệ để tìm kiếm
    scale_steps=5,  # Số bước tỷ lệ
    rotation_range=(0, 0),  # Phạm vi góc xoay (độ)
    rotation_steps=1,  # Số bước góc xoay
    use_gray=True,  # Sử dụng ảnh xám để tăng tốc độ
    use_canny=False  # Sử dụng phát hiện cạnh Canny cho hình ảnh mờ
)

if result:
    x, y, confidence = result
    print(f"Đã tìm thấy hình ảnh tại ({x}, {y}) với độ tương đồng {confidence}")
else:
    print("Không tìm thấy hình ảnh")

Nhấp vào hình ảnh

Tìm và nhấp vào hình ảnh mẫu trên màn hình thiết bị.

# Tìm và nhấp vào hình ảnh
success = image_interaction.tap_image(
    template_path="/path/to/button.png",
    threshold=0.8,
    tap_offset=(0, 0)  # Độ lệch (dx, dy) khi nhấp so với tâm hình ảnh
)

if success:
    print("Đã nhấp vào hình ảnh thành công")
else:
    print("Không tìm thấy hình ảnh để nhấp")

Đợi và nhấp hình ảnh

Đợi hình ảnh xuất hiện trên màn hình trong một khoảng thời gian và nhấp vào khi tìm thấy.

# Đợi hình ảnh xuất hiện và nhấp vào
success = image_interaction.wait_and_tap_image(
    template_path="/path/to/button.png",
    timeout=10,      # Thời gian tối đa đợi (giây)
    interval=0.5,    # Khoảng thời gian giữa các lần tìm kiếm (giây)
    threshold=0.8    # Ngưỡng tương đồng
)

if success:
    print("Đã đợi và nhấp vào hình ảnh thành công")
else:
    print("Hình ảnh không xuất hiện trong thời gian chờ")

# Kiểm tra xem hình ảnh có xuất hiện không
is_present = image_interaction.is_image_present("/path/to/element.png")
print(f"Hình ảnh có mặt: {is_present}")

Kéo thả hình ảnh

Kéo từ hình ảnh nguồn đến hình ảnh đích trên màn hình.

# Kéo từ hình ảnh này đến hình ảnh khác
success = image_interaction.drag_image_to_image(
    source_template_path="/path/to/source.png",
    target_template_path="/path/to/target.png",
    duration=800,    # Thời gian kéo (ms)
    threshold=0.8    # Ngưỡng tương đồng
)

if success:
    print("Đã kéo thành công từ nguồn đến đích")
else:
    print("Không thể kéo vì không tìm thấy một hoặc cả hai hình ảnh")

Tùy chọn nâng cao

OIADB cung cấp nhiều tùy chọn nâng cao cho nhận diện hình ảnh để xử lý các trường hợp phức tạp.

Tùy chọn tỷ lệ và xoay

Tìm kiếm hình ảnh với các tỷ lệ và góc xoay khác nhau để xử lý các trường hợp khi hình ảnh mẫu có kích thước hoặc hướng khác với màn hình.

# Tìm hình ảnh với phạm vi tỷ lệ và góc xoay
result = image_interaction.find_image(
    template_path="/path/to/template.png",
    scale_range=(0.5, 1.5),    # Tìm từ 50% đến 150% kích thước gốc
    scale_steps=10,            # Số bước tỷ lệ
    rotation_range=(-30, 30),  # Tìm từ -30 đến +30 độ
    rotation_steps=6           # Số bước góc xoay
)

Xử lý hình ảnh mờ

Sử dụng phát hiện cạnh Canny để cải thiện nhận diện hình ảnh mờ hoặc có độ tương phản thấp.

# Sử dụng phát hiện cạnh Canny
result = image_interaction.find_image(
    template_path="/path/to/template.png",
    use_canny=True,  # Bật phát hiện cạnh Canny
    threshold=0.6    # Có thể sử dụng ngưỡng thấp hơn với Canny
)

XML Dump & Trợ năng

Module XML Dump của OIADB cung cấp các phương thức để truy xuất cấu trúc XML của giao diện người dùng Android và tương tác thông qua API trợ năng.

Lấy cấu trúc UI

Truy xuất cấu trúc XML của giao diện người dùng hiện tại trên thiết bị.

from oiadb.commands import xml_dump

# Lấy cấu trúc XML của giao diện người dùng hiện tại
xml_tree = xml_dump.get_ui_xml()

# In ra cấu trúc XML
print(xml_dump.format_xml(xml_tree))

# Lưu cấu trúc XML vào file
xml_dump.save_xml(xml_tree, "/path/to/output.xml")

Tìm phần tử

Tìm kiếm các phần tử trong cấu trúc UI dựa trên văn bản, thuộc tính hoặc vị trí.

# Tìm phần tử theo văn bản
element = xml_dump.find_element_by_text("Login")
if element:
    print(f"Đã tìm thấy phần tử: {xml_dump.get_element_info(element)}")

# Tìm phần tử theo thuộc tính
element = xml_dump.find_element_by_attribute("resource-id", "com.example.app:id/username")
if element:
    print(f"Đã tìm thấy phần tử: {xml_dump.get_element_info(element)}")

# Tìm phần tử theo vị trí
element = xml_dump.find_element_at_position(500, 800)
if element:
    print(f"Đã tìm thấy phần tử: {xml_dump.get_element_info(element)}")

# Tìm nhiều phần tử theo thuộc tính
elements = xml_dump.find_elements_by_attribute("class", "android.widget.Button")
print(f"Đã tìm thấy {len(elements)} nút")

Tương tác với phần tử

Tương tác với các phần tử UI thông qua API trợ năng.

# Nhấp vào phần tử
element = xml_dump.find_element_by_text("Login")
if element:
    xml_dump.click_element(element)

# Nhập văn bản vào phần tử
element = xml_dump.find_element_by_attribute("resource-id", "com.example.app:id/username")
if element:
    xml_dump.input_text(element, "username@example.com")

# Xóa văn bản trong phần tử
xml_dump.clear_text(element)

# Cuộn đến phần tử
element = xml_dump.find_element_by_text("Terms of Service")
if element:
    xml_dump.scroll_to_element(element)

# Kiểm tra trạng thái phần tử
is_checked = xml_dump.is_element_checked(element)
is_enabled = xml_dump.is_element_enabled(element)
is_focused = xml_dump.is_element_focused(element)

XML Server

Khởi động và quản lý XML Server để cải thiện hiệu suất khi truy xuất cấu trúc UI nhiều lần.

# Khởi động XML Server
server, server_thread = xml_dump.start_xml_server(port=8000)

# Thiết lập port forwarding
xml_dump.setup_port_forwarding(device_port=8000, host_port=8000)

# Sử dụng XML Server để lấy cấu trúc UI
xml_tree = xml_dump.get_ui_xml_from_server()

# Dừng XML Server khi hoàn tất
xml_dump.stop_xml_server(server)

Kiến trúc thư viện

OIADB được thiết kế với kiến trúc module hóa, cho phép mở rộng dễ dàng và tích hợp liền mạch với các dự án Python khác.

Module cốt lõi

  • MyADB: Lớp chính cung cấp giao diện để tương tác với ADB
  • adb_manager: Quản lý việc tải xuống, cài đặt và chạy ADB
  • exceptions: Định nghĩa các ngoại lệ cụ thể của thư viện

Module lệnh

  • interaction: Các lệnh tương tác cơ bản với thiết bị
  • image_interaction: Tương tác dựa trên nhận diện hình ảnh
  • xml_dump: Truy xuất và phân tích cấu trúc UI
  • apps: Quản lý ứng dụng (cài đặt, gỡ cài đặt, khởi động)
  • file_ops: Thao tác với hệ thống tệp trên thiết bị
  • connect: Kết nối và quản lý thiết bị

Module tiện ích

  • platform_utils: Tiện ích đa nền tảng
  • image_recognition: Thuật toán nhận diện hình ảnh
  • logger: Hệ thống ghi nhật ký
  • config: Quản lý cấu hình
