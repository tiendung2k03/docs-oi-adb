Bắt đầu với OIADB

Yêu cầu

Trước khi bắt đầu với OIADB, hãy đảm bảo hệ thống của bạn đáp ứng các yêu cầu sau:

  • Python 3.6 trở lên
  • Không cần cài đặt ADB thủ công: Thư viện sẽ tự động tải xuống và cài đặt ADB nếu cần
  • Lưu ý: Thư viện sẽ tự động cài đặt và quản lý oiadb-server.apk trên thiết bị Android của bạn.
  • Tùy chọn (cho nhận dạng ảnh): OpenCV và NumPy (cần cài đặt riêng, xem phần Cài đặt).

Lưu ý

Để sử dụng đầy đủ tính năng của OIADB, thiết bị Android của bạn cần được bật chế độ USB Debugging. Xem hướng dẫn bật USB Debugging tại đây.

Cài đặt

Cài đặt cơ bản

Cách đơn giản nhất để cài đặt OIADB là thông qua pip:

pip install oiadb

Cài đặt với hỗ trợ nhận diện hình ảnh

Để sử dụng tính năng nhận diện hình ảnh, bạn cần cài đặt với các phụ thuộc bổ sung:

pip install oiadb[image]

Cài đặt từ mã nguồn

Bạn cũng có thể cài đặt OIADB trực tiếp từ mã nguồn:

git clone https://github.com/tiendung102k3/oiadb
cd oiadb
pip install -e .

Cài đặt trên Termux

Để cài đặt OIADB trên Termux (Android):

pkg install python
pip install oiadb

Sử dụng cơ bản

Khởi tạo

Để bắt đầu sử dụng OIADB, bạn cần khởi tạo đối tượng MyADB:

from oiadb import MyADB

# Khởi tạo với thiết bị mặc định
adb = MyADB()

# Hoặc chỉ định thiết bị cụ thể
adb = MyADB(device_id="emulator-5554")

# Tắt tự động cài đặt ADB (nếu bạn đã cài đặt ADB)
adb = MyADB(auto_install_adb=False)

Các thao tác cơ bản

Dưới đây là một số thao tác cơ bản bạn có thể thực hiện với OIADB:

# Liệt kê các thiết bị đã kết nối
devices = adb.get_devices()
print(devices)

# Cài đặt ứng dụng
adb.install_app("/path/to/app.apk")

# Gỡ cài đặt ứng dụng
adb.uninstall_app("com.example.app")

# Khởi động lại thiết bị
adb.reboot_device()

Tương tác với thiết bị

OIADB cung cấp nhiều cách để tương tác với thiết bị Android:

from oiadb.commands import interaction

# Nhấn vào tọa độ màn hình
interaction.tap(500, 500)

# Vuốt màn hình
interaction.swipe(100, 500, 600, 500)

# Nhập văn bản
interaction.text_input("Hello World")

# Nhấn phím Home
interaction.home()

Hướng dẫn đa nền tảng

Windows

  1. Cài đặt Python: Tải và cài đặt Python từ python.org
  2. Cài đặt OIADB: 
    pip install oiadb[image]
  3. Sử dụng: 
    from oiadb import MyADB
adb = MyADB()  # Tự động tải và cài đặt ADB nếu cần

Lưu ý cho Windows: Nếu gặp lỗi về đường dẫn, hãy sử dụng dấu gạch chéo ngược kép (\\) hoặc dấu gạch chéo thuận (/).

macOS

  1. Cài đặt Python: Sử dụng Homebrew hoặc tải từ python.org
  2. Cài đặt OIADB: 
    pip3 install oiadb[image]
  3. Sử dụng: 
    from oiadb import MyADB
adb = MyADB()

Lưu ý cho macOS: Có thể cần cấp quyền cho ADB trong Cài đặt Bảo mật & Quyền riêng tư.

Linux

  1. Cài đặt Python: Sử dụng trình quản lý gói của bản phân phối
  2. Cài đặt OIADB: 
    pip3 install oiadb[image]
  3. Sử dụng: 
    from oiadb import MyADB
adb = MyADB()

Lưu ý cho Linux: Có thể cần cài đặt các gói phụ thuộc bổ sung cho OpenCV.

Termux

  1. Cài đặt Python: 
    pkg install python
  2. Cài đặt OIADB: 
    pip install oiadb
  3. Sử dụng: 
    from oiadb import MyADB
adb = MyADB(auto_install_adb=False)  # Sử dụng ADB đã cài đặt

Lưu ý cho Termux: Một số tính năng có thể bị hạn chế khi chạy trên Android.

Xử lý sự cố

Vấn đề cài đặt

Đảm bảo bạn có quyền ghi vào thư mục cài đặt. Trên Linux/macOS, có thể cần chạy với sudo.

sudo pip install oiadb

Hoặc bạn có thể tự cài đặt ADB và tắt tính năng tự động cài đặt:

adb = MyADB(auto_install_adb=False)

Vấn đề kết nối

Đảm bảo USB debugging đã được bật trên thiết bị Android và thiết bị đã được cấp quyền.

  1. Kiểm tra kết nối USB
  2. Đảm bảo đã bật USB debugging trong Developer Options
  3. Chấp nhận lời nhắc "Allow USB debugging" trên thiết bị
  4. Kiểm tra thiết bị được phát hiện bằng lệnh: 
    from oiadb import MyADB
adb = MyADB()
devices = adb.get_devices()
print(devices)

Vấn đề phụ thuộc

Đảm bảo bạn đã cài đặt các phụ thuộc cần thiết:

pip install opencv-python numpy

Nếu gặp lỗi khi cài đặt OpenCV, hãy thử cài đặt phiên bản headless:

pip install opencv-python-headless
