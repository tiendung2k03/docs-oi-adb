OIADB - Python Wrapper cho Android Debug Bridge

Thư viện Python mạnh mẽ giúp tự động hóa và tương tác với thiết bị Android thông qua ADB với các tính năng nâng cao như nhận diện hình ảnh và XML dump.

Bắt đầu ngay GitHub 
pip install oiadb

Tính năng chính

Tự động cài đặt ADB

Tự động tải xuống và cài đặt ADB nếu cần, không yêu cầu cài đặt thủ công.

Nhận diện hình ảnh

Tìm kiếm và tương tác với các phần tử trên màn hình dựa trên hình ảnh mẫu.

XML Dump & Trợ năng

Truy xuất cấu trúc UI và tương tác với các phần tử thông qua API trợ năng.

Quản lý thiết bị

Dễ dàng kết nối và quản lý nhiều thiết bị Android, bao gồm cả kết nối không dây.

Đa nền tảng

Hỗ trợ Windows, macOS, Linux và thậm chí cả Termux trên Android.

API đơn giản

Giao diện lập trình đơn giản và trực quan giúp tự động hóa các tác vụ phức tạp.

Kiến trúc OIADB

OIADB được thiết kế với kiến trúc module hóa, cho phép mở rộng dễ dàng và tích hợp liền mạch với các dự án Python khác.

  • Lớp MyADB cung cấp giao diện chính để tương tác với ADB
  • Module commands chứa các lệnh được nhóm theo chức năng
  • Module image_recognition xử lý tìm kiếm và tương tác dựa trên hình ảnh
  • Module xml_dump quản lý truy xuất và phân tích cấu trúc UI
  • Module utils cung cấp các tiện ích và hàm trợ giúp

Bắt đầu nhanh

# Cài đặt cơ bản
pip install oiadb

# Cài đặt với hỗ trợ nhận diện hình ảnh
pip install oiadb[image]

# Cài đặt từ mã nguồn
git clone https://github.com/tiendung102k3/oiadb
cd oiadb
pip install -e .
from oiadb import MyADB

# Khởi tạo với thiết bị mặc định
adb = MyADB()

# Hoặc chỉ định thiết bị cụ thể
adb = MyADB(device_id="emulator-5554")

# Liệt kê các thiết bị đã kết nối
devices = adb.get_devices()
print(devices)

# Cài đặt ứng dụng
adb.install_app("/path/to/app.apk")

# Gỡ cài đặt ứng dụng
adb.uninstall_app("com.example.app")
from oiadb.commands import image_interaction

# Tìm hình ảnh đơn
result = image_interaction.find_image(
    template_path="/path/to/template.png",
    threshold=0.8,  # Ngưỡng tương đồng (0.0 - 1.0)
    region=None,    # Vùng tìm kiếm (x, y, width, height), None = toàn màn hình
    scale_range=(0.8, 1.2),  # Phạm vi tỷ lệ để tìm kiếm
    scale_steps=5,  # Số bước tỷ lệ
    rotation_range=(0, 0),  # Phạm vi góc xoay (độ)
    rotation_steps=1,  # Số bước góc xoay
    use_gray=True,  # Sử dụng ảnh xám để tăng tốc độ
    use_canny=False  # Sử dụng phát hiện cạnh Canny cho hình ảnh mờ
)

if result:
    x, y, confidence = result
    print(f"Đã tìm thấy hình ảnh tại ({x}, {y}) với độ tương đồng {confidence}")
else:
    print("Không tìm thấy hình ảnh")
from oiadb.commands import xml_dump

# Lấy cấu trúc XML của giao diện người dùng hiện tại
xml_tree = xml_dump.get_ui_xml()

# Tìm phần tử theo văn bản
element = xml_dump.find_element_by_text("Login")
if element:
    # Nhấp vào phần tử
    xml_dump.click_element(element)

# Tìm phần tử theo thuộc tính
element = xml_dump.find_element_by_attribute("resource-id", "com.example.app:id/username")
if element:
    # Nhập văn bản vào phần tử
    xml_dump.input_text(element, "username@example.com")

Hỗ trợ đa nền tảng

Windows

Hỗ trợ đầy đủ trên Windows với tự động cài đặt ADB và tất cả các tính năng.

macOS

Tương thích hoàn toàn với macOS, bao gồm cả Intel và Apple Silicon.

Linux

Hỗ trợ các bản phân phối Linux phổ biến với hiệu suất tối ưu.

Termux

Chạy trực tiếp trên thiết bị Android thông qua Termux với các tính năng cốt lõi.
